Projects

jGrader

Description:

jGrader is a cloud based grading system designed for use within a high school AP Computer Science program. The system allows teachers to easily collect, grade, and provide feedback on assignments. Teachers can execute code submitted by their students directly in their browsers. Over 120 teachers use the website.

How I built this website:

jGrader.com uses a node.js express stack, the database is mySQL and the frontend uses the EJS templating engine with standard javascript (jquery) and CSS. The private git repository is on www.github.com. I have spent over 200 hours on this project.

Process Watch

Description:

Most recently, an IT professional and family friend mentioned that he could not find software to effectively track his clients' backups and replications. So I created a website called Process Watch (www.process.watch), which monitors backups and replications by automatically parsing emails sent by client hardware systems. Currently we have eight "beta testers" and the website parses over 50,000 emails a month. By January 2016, I plan to offer it as a commercial product.

How I built this website:

The most difficult aspect of this project is optomizing the website. Because the server has to parse thousands of emails
